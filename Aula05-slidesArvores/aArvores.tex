\begin{Slide}{Introdução - Árvore}
Um conjunto de nós tal que:  
\begin{itemize}
  \item existe um nó $r$, denominado \emph{raiz}, com zero ou mais
    sub-árvores, cujas raízes estão ligadas a $r$.

\begin{center}
    \includegraphics[width=0.8\textwidth]{figuras/fig01.pdf}
\end{center}

\newpage

  \item os nós raízes destas sub-árvores são os \emph{filhos} de $r$.
  \item os \emph{nós internos} da árvore são os nós com filhos.
  \item as \emph{folhas} ou \emph{nós externos} da árvore são os nós
    sem filhos. 
  \end{itemize}
\end{Slide}

\begin{Slide}{Árvores Binárias}
\begin{itemize}
\item uma árvore em que cada nodo tem zero, um ou dois filhos.
\item uma árvore binária é:
\begin{itemize}
\item uma árvore vazia, ou; 
\item um nodo raiz com duas sub-árvores: a sub-árvore da direita
  (sad); a sub-árvore da esquerda (sae).
\end{itemize}
\end{itemize}
\begin{center}
    \includegraphics[width=1\textwidth]{figuras/fig02.pdf}
\end{center}
\end{Slide}


\begin{Slide}{Árvores Binárias - Exemplo}
Árvore binária representando expressões aritméticas:
\begin{itemize}
\item nós folhas representam operandos
\item nós internos operadores
\item exemplo: $(3+6) \times (4-1)+5$
\end{itemize}

\begin{Slide}{Altura de árvore binárias}
\begin{itemize}
\item Propriedade fundamental de árvores: só existe um caminho da raiz
  para qualquer nó.
\item Altura de uma árvore: comprimento do caminho mais longo da raiz
  até uma das folhas.
\begin{itemize}
\item a altura de uma árvore com um único nó raiz é zero
\item a altura de uma árvore vazia é -1
\end{itemize}
\end{itemize}
\end{Slide}

\begin{Slide}{Altura de árvore binárias}
\begin{itemize}
\item Nível de um nó:
\begin{itemize}
\item a raiz está no nível 0, seus filhos diretos estão no nível 1, ...
\item o último nível da árvore é a altura da árvore.
\end{itemize}
\end{itemize}
\end{Slide}

\begin{Slide}{Altura de árvore binárias}
\begin{itemize}
\item Árvore cheia:
\begin{itemize}
\item todos os seus nós internos tem duas sub-árvores associadas. 
\item número $n$ de nós de uma árvore cheia de altura $h$ é igual a
\begin{equation}
n = s^{k+1} - 1
\end{equation}
\end{itemize}
\end{itemize}
\begin{figure}[htbp]

\end{Slide}

\begin{Slide}{Altura de árvore binárias}
\begin{itemize}
\item Árvore degenerada:
\begin{itemize}
\item todos os seus nós internos tem uma única sub-árvore associada. 
\item número $n$ de nós de uma árvore cheia de altura $h$ é igual a
\begin{equation}
n = h + 1
\end{equation}
\end{itemize}
\end{itemize}

\begin{Slide}{Topologia Binária}
\begin{itemize}
\item Representação de um nodo de uma árvore n-ária $\equiv$
  representação de um nodo de uma árvore binária. 
\item Um nodo possui a informação que deve ser armazenada e duas
  referências para sub-árvores:
\begin{itemize}
\item árvore binária: referências para as sub-árvores à esquerda e à direita. 
\item árvore n-ária: referências para a primeira sub-árvore filha e
  para a sub-árvore irmã. 
\end{itemize}
\end{itemize}
\end{Slide}
