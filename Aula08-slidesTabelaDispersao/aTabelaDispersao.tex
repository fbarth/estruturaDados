\begin{Slide}{Tabela de dispersão}
Os objetivos de uma tabela de dispersão são:
\begin{itemize}
\item recuperar um elemento em tempo constante $O(1)$, e;
\item armazenar o índice utilizando uma estrutura otimizada - que
  ocupa pouco espaço.
\end{itemize}
\end{Slide}


\begin{Slide}{Exemplo de uso de tabela de dispersão}

\textit{``... considere uma situação onde você deve desenvolver um
  sistema que armazena informações sobre alunos. Cada registro de
  aluno, além das informações sobre nome e email, também terá um
  índice - que deverá facilitar a busca do registro... ``}

\begin{itemize}
\item Que estrutura utilizar para implementar o armazenamento e busca
  dos registros de forma otimizada?

\newpage

\begin{itemize}
\item Podemos utilizar um vetor de tamanho $N$ (que respeita a faixa
  de valores do índice).
\item Digamos que o índice tem 7 dígitos, então teríamos que ter um
  vetor com 10 milhões de posições.
\item Com $Aluno a = alunos[i]$ temos:

\begin{itemize}
\item um acesso rápido;
\item mas pagamos um preço em uso de memória proibitivo.
\end{itemize}

\end{itemize}

\end{itemize}
\end{Slide}
